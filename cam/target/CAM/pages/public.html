CAM

CAM is a web application which provides a user interface for exploiting the CAM-Service API interactively. CAM allows user to create, read, update and delete Classes, Models, Assets, Attributes, Relationships and Owners through a web interface.

CAM-Service

CAM-Service is a web application with no user interface. It provides a REST-based abstraction on top of the reference ontology that is stored in the RDF4J repository (not included: see installation instructions below). By means of API calls, applications can query the reference ontology: CRUD operations are allowed on Classes, Models, Assets, Attributes, Relationships and Owners. CAM-Service also implements the integration with the FIWARE Orion Context Broker and with FIWARE security.
